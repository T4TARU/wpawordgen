# Compile
```bash
make app
```

# Install
```bash
sudo make install
```

# Unintall
```bash
sudo make uninstall
```

# Dependencies
- `build-essential`
- `make`

# Install Dependencies
```bash
sudo apt update
```

```bash
sudo apt dist-upgrade -y
```

```bash
sudo apt install -y build-essential make
```
